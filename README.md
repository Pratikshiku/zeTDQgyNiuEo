## 闲置物品交易系统

### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 根据名称和分类检索
5. 二手物品详情页
6. 评论区
7. 个人信息
8. 发布个人闲置物品
9. 购物车
10. 修改密码
11. 安全退出


**管理员功能：**
1. 登录
2. 首页数据大屏
3. 菜单管理
4. 角色管理
5. 用户管理
6. 日志管理
7. 监控管理
8. 商品类目管理
9. 商品管理
10. 学生管理
11. 评论管理
12. 首页新闻管理
13. 网站设置
14. 安全退出
